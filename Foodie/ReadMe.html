Furniture Haven Setup Instructions

Follow the steps below to set up the Furniture Haven application on your local machine:

  1. Unzip the folder containing the application files.
  2. Backup the database file named Furniture_havenDB.bak located in the database folder.
  3. Change the data source in the web.config file. If necessary, modify the username and password for SQL Server Management Studio (SSMS) or change it to Integrated Security=True;.
  4. Open and run the .sln file in Visual Studio.
  5. For admin access, use the following credentials:
    • Username: Admin
    • Password: 123
  6. For user access, you can either register a new user or use the following credentials:
    • Username: a
    • Password: 123
